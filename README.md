# Starten

Kommandozeile:

```
git clone git@github.com:maltesa/piano.git
cd piano
yarn install
yarn dev
```

http://localhost:3000/ im Browser aufrufen.

# template-vite-react-tailwind-v3

- **template of:** vite + react
- **tailwindcss:** v3.x

## Started

```bash
yarn
# or `npm install`
```

---

## Develop

```bash
yarn dev
# or `npm run dev`
```

---

## Build

```bash
yarn build
# or `npm run build`
```
